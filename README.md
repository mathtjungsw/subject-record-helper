# 나만의 교과세특 기재도우미

교사가 학생 자료와 간단 평가를 바탕으로 ChatGPT에 붙여넣을 교과세특 작성 프롬프트를 만드는 단일 HTML 웹툴입니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 기본 설정, 참고 사례, 학생 자료, 교사 평가를 입력합니다.
   - txt, md, csv, pdf 업로드는 내용 추출을 시도하고 문서 안의 헤더를 기준으로 여러 자료 카드로 나눕니다.
   - PDF 추출은 브라우저에서 PDF.js를 불러와 처리하며, 파일 내용은 서버로 업로드하지 않습니다.
3. `프롬프트 생성` 후 복사해서 ChatGPT에 붙여넣습니다.
4. 생성된 세특 결과를 다시 붙여넣고 학생별로 저장합니다.
5. 다른 PC에서 이어서 작업하려면 `세이브 파일 저장`으로 JSON을 내려받고, 새 PC에서 `세이브 파일 불러오기`를 사용합니다.

모든 데이터는 브라우저 localStorage 또는 사용자가 내려받는 JSON/CSV 파일에만 저장됩니다. 별도 서버나 DB를 사용하지 않습니다.
