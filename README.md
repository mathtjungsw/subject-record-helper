# WORKBOX 개인 업무 도구

`index.html`을 열면 이 폴더의 개인용 웹도구를 사용할 수 있습니다. 서버 없이 브라우저에서 동작하며, 직접 입력한 데이터는 현재 브라우저의 로컬 저장소에만 보관됩니다.

## 경남 중등 인사이력

- 수학교사, 교감, 교장, 장학사, 장학관, 기관장의 이름·학교·교육지원청 검색
- 제공된 2023.3.1, 2024.3.1, 2024.9.1, 2025.3.1, 2026.3.1 인사자료 반영
- 최근 자료의 신임지를 ‘최근 확인 근무지’로 표시
- 발령 기준일별 신임지·현임지·직위·원본 파일과 쪽수 표시
- 직접 입력, CSV 일괄 입력, JSON 백업·복원, 검색결과 CSV 저장

‘최근 확인 근무지’는 제공된 인사이동 자료만으로 계산한 값입니다. 이후 발령 자료가 없거나 동명이인이 있으면 실제 현 근무지와 다를 수 있으므로 원본 출처를 함께 확인해야 합니다.

## 인사자료 갱신

새로운 3월·9월 PDF/HWP 자료를 원본 자료 폴더에 넣은 뒤 아래 변환기를 실행하면 `tools/personnel-data.js`가 갱신됩니다. 스캔 PDF는 Windows 한글 OCR을 사용하고 HWP는 표 구조를 읽습니다.

```powershell
python scripts/build-personnel-data.py `
  --source "N:\개인\자주 찾아보는 파일\중등인사발표" `
  --output "tools\personnel-data.js" `
  --cache "tmp\personnel-data-cache" `
  --pyhwp ".codex_work\pyhwp"
```

일반 사용 중에는 이 명령을 실행할 필요가 없습니다. 새 인사 원본을 Codex에 전달해 갱신을 요청해도 됩니다.

## 교과세특 기재도우미

학생 자료와 교사 평가를 정리해 세특 작성용 프롬프트와 결과를 관리하는 별도 도구입니다. 자세한 사용 방법은 도구 화면의 안내를 따릅니다.
